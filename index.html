ABOUT

Modular Monday is an event hosted in VRChat every Monday at 7:00pm CST where people create music live with Euro rack, hardware, software and other instruments.

HOW TO JOIN EVENTS

Modular Monday VRChat Group

Join our VRChat Group by clicking the button above. If you don't already have a VRC account, create one then join the group. When an event starts a notification in-game will pop up!

Be sure to click the Discord icon at the top of this page to join our Discord server, especially if you need help or have any questions! We have a great community that loves to help

HOW TO PERFORM

  • Join the VRC group
  • Join the Discord
  • Come out to an event to hang and see how it all works!

There is an opentable at the end for newcomers to perform. More info can be found in our Discord.
